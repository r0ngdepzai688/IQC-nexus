# 07. Dashboard Layer

Dashboards in the New Models module are tailored to user roles to provide immediate context and actionable insights.

## Available Dashboards

### Executive Dashboard
- **Target Audience:** Team Leader, Admin.
- **Focus:** High-level project health, Overall RAG status, Critical Risks, MP Readiness.
- **Widgets:** Portfolio Timeline, Aggregated KPIs, Risk Escalation Summary.

### Cell Leader Dashboard
- **Target Audience:** Cell Leader.
- **Focus:** Daily operations, incoming lots, IQC resource allocation.
- **Widgets:** Today's Tasks, Pending IQC Results, Open Issues.

### Engineer Dashboard (Part Leader)
- **Target Audience:** Part Leader, Staff.
- **Focus:** BOM progress, Vendor capability, Documentation status.
- **Widgets:** Missing ISS list, Pending Golden Samples, DFx Status, Vendor Gap Analysis.

### Project Dashboard
- **Target Audience:** Group Leader.
- **Focus:** Deep dive into a specific New Model project.
- **Widgets:** Phase Completion (PV/PR/SR/MP), Project Timeline, Task Burn-down.

### Vendor Dashboard
- **Target Audience:** Part Leader, Group Leader.
- **Focus:** Supplier performance across models.
- **Widgets:** Vendor Scorecard, Audit History, CPK/Yield Trends.

### Risk Dashboard
- **Target Audience:** All Roles (Filtered by scope).
- **Focus:** Risk mitigation tracking.
- **Widgets:** Risk Matrix (Likelihood vs. Impact), Aging Risks, Action Item Status.

### KPI Dashboard
- **Target Audience:** Management.
- **Focus:** Detailed metrics.
- **Widgets:** Historical Trends, RAG indicators, Target vs. Actual charts.

### Timeline Dashboard
- **Target Audience:** Project Managers.
- **Focus:** Schedule adherence.
- **Widgets:** Gantt chart, Milestone Tracker, Delay predictions.

### Action Center
- **Target Audience:** All Roles.
- **Focus:** Personal task management.
- **Widgets:** My Tasks, Pending Approvals, Unread Notifications.
