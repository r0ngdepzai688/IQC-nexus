# Business Knowledge Base

This directory contains the permanent business documentation for the Portal platform.
This is the single source of truth for business rules, workflows, and definitions.

## Principles
- Never hardcode business logic.
- Business rules must come from these documents.
- Every new feature must first update documentation.
- Documentation is always newer than code.
- When requirements conflict: stop coding, ask for clarification, update documentation, then continue implementation.
- Always preserve backward compatibility.
- Never delete documentation.
